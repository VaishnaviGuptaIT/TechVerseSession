Code Comparison: Media Query vs Tailwind CSS

Same responsive card, two different approaches

Using Media Queries (Pure CSS)

/* Pure CSS with Media Queries */
.card {
  background: white;
  padding: 24px;
  border-radius: 12px;
  width: 100%;
}

/* Tablet */
@media (min-width: 768px) {
  .card {
    width: 50%;
    padding: 32px;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .card {
    width: 33.33%;
    padding: 40px;
  }
}

Using Tailwind CSS

<!-- Tailwind CSS Classes -->
<div class="w-full p-6 rounded-xl bg-white
            md:w-1/2 md:p-8
            lg:w-1/3 lg:p-10">
  Card Content
</div>

/* Breakpoints:
   sm: 640px
   md: 768px
   lg: 1024px
   xl: 1280px */

Key Difference: Media queries require writing separate CSS blocks, while Tailwind uses utility classes directly in HTML for faster development.

React Basics

What is React and how it works

⚛️ What is React?

React is a JavaScript library for building user interfaces. It lets you compose complex UIs from small and isolated pieces of code called "components."

🏗️ React Structure

React applications are built as a tree of components. Each component is a self-contained piece of UI that can have its own state and logic.

🎯 Why We Use React?

React makes it easy to create interactive UIs. It efficiently updates and renders the right components when your data changes. Declarative views make code predictable and easier to debug.

⚙️ How We Use React?

We create functional components using JSX (JavaScript XML), use hooks for state management, and compose components together to build complex applications.

Simple React Example

A counter component with live preview

Counter Component 
// React Counter Component
import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div className="counter">
      <h2>Counter: {count}</h2>
      <button onClick={() => setCount(count + 1)}>
        Increment
      </button>
      <button onClick={() => setCount(count - 1)}>
        Decrement
      </button>
      <button onClick={() => setCount(0)}>
        Reset
      </button>
    </div>
  );

export default Counter;

    <button onclick="appendToDisplay('*')">*</button>
    <button onclick="deleteLast()">←</button>
    <button onclick="appendToDisplay('7')">7</button>
    <button onclick="appendToDisplay('8')">8</button>
    <button onclick="appendToDisplay('9')">9</button>
    <button onclick="appendToDisplay('-')">-</button>

let display = document.getElementById('display');

function appendToDisplay(value) {
  display.value += value;
}

function clearDisplay() {
  display.value = '';
}

function deleteLast() {
  display.value = display.value.slice(0, -1);
}

function calculate() {
  try {
    display.value = eval(display.value);
  } catch (error) {
    display.value = 'Error';
  }
}
